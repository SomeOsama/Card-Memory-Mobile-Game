# Ace!: A Card Memorization Mobile Game
Development Details:
- Unity Version:
- Windows 11
- Testing Devices: Google Pixel 6,

Game Instructions:
The goal of the game is to match the cards as fast as possible. There will be power-ups and hints available based on the level difficulty, whether the user watches ads, etc.

Controls:
- Touch/Tap
- Swipe
- Pinch zoom
